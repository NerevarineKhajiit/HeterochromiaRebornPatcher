Heterochromia Reborn Patcher

Removing eye records from the vanilla slider prevents issues with overlapping eyes. If you want to leave them in the slider anyway, uncheck both boxes.
Leave Modded eye sets unchecked and just disable the original mods' .esp files to save plugin room. Only check this if you need the original .esp for something like a follower mod.
You can have the patcher ignore Vanilla texture sets if you only want to have modded eyes show up in the new sliders, or if you want this to be an addon plugin that you can distribute - see instructions doc for more on this.
